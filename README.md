# Linux Password Strength Analyzer

This project is a Python-based password strength analyzer created for CYB-333 Security Automation.

## Project Objective

The purpose of this tool is to evaluate passwords against basic Linux security and STIG-style password requirements. The program checks password length, uppercase letters, lowercase letters, numbers, and special characters. It then provides a score, strength rating, and recommendations for improvement.

## Features

- Checks minimum password length
- Checks for uppercase letters
- Checks for lowercase letters
- Checks for numbers
- Checks for special characters
- Calculates a password score
- Provides a strength rating
- Gives recommendations for weak passwords

## Requirements

- Python 3.x

## How to Run

```bash
python password_strength_analyzer.py
